# Tanya Topacharova — Portfolio

A static portfolio site showcasing selected projects: full-stack web apps, AI automations, and mobile apps.

- `index.html` — the single-page site (no build step)
- `images/` — project screenshots
- `Tanya_Topacharova_Portfolio.pdf` — downloadable PDF version

## Run locally

```bash
python3 -m http.server 4180
# open http://localhost:4180
```

## Deploy

Static site — no build. On Cloudflare Pages use **Build command:** *(none)* and **Output directory:** `/`.
